Overview

Pjax is a technique that combines Ajax and history.pushState to speed up page transitions on classic multi-page websites.

It lets you combine server-side rendering and routing with fast clientside navigation.

Open your browser's network console and observe it while navigating the site. This site is very lightweight, so the effects won't seem very dramatic. However, for heavy websites with large amounts of contents, the improvements can be cataclysmic.

See the full readme in the source repository .
